## Contributing

1. Fork the project on GitHub.
2. Clone the repository:
   ```sh
   git clone https://github.com/<username>/spoti
   ```
3. Install dependencies:
   ```sh
   nvm use && yarn install
   ```
4. Create a global `npm` link:
   ```sh
   npm link
   ```
5. Happy developing!
