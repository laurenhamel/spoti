# Spotify URL Formats

| Type     | Format                                   |
| -------- | ---------------------------------------- |
| Track    | `https://open.spotify.com/track/<id>`    |
| Album    | `https://open.spotify.com/album/<id>`    |
| Playlist | `https://open.spotify.com/playlist/<id>` |
| Artist   | `https://open.spotify.com/artist/<id>`   |
